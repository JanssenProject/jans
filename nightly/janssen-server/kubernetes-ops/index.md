# Kubernetes Operation Guide

This Operation guide helps you learn about the common operations for Janssen on Kubernetes.

## Prerequisite

- Review the [Janssen](https://docs.jans.io/nightly/janssen-server/reference/kubernetes/index.md) components and architecture
- Complete the [installation](https://docs.jans.io/nightly/janssen-server/install/helm-install/index.md) of Janssen on your desired Kubernetes provider.
