# Using External Libraries In Interception Scripts

Janssen Server [Interception script](https://docs.jans.io/nightly/janssen-server/developer/scripts/index.md) is a powerful tool that enables developers to control many different aspects of the authentication process.

It is possible to use external Java libraries in [pure Java](https://docs.jans.io/nightly/janssen-server/developer/scripts/#using-java-libraries-in-a-script) based interception scripts. If the interception script is implemented in Jython then it is possible to [use external Java libraries](https://docs.jans.io/nightly/janssen-server/developer/scripts/#using-java-libraries-in-a-jython-script) as well as [external Python libraries](https://docs.jans.io/nightly/janssen-server/developer/scripts/#using-python-libraries-in-a-script)
