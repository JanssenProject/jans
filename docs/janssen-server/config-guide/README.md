# Configure Identity & Policies

This section covers configuring your Janssen deployment, including identity management, access policies, and customization options.

## Configuration Tools

Janssen provides multiple tools for configuration:

- **TUI (Text-Based UI)** - Interactive terminal interface for configuration
- **CLI (Command Line)** - Scriptable command-line configuration tool
- **REST API (Config API)** - Programmatic configuration via REST endpoints

## Key Configuration Areas

- **Clients** - OAuth 2.0/OpenID Connect client registration and management
- **Scopes** - Define and manage permission scopes
- **Users & Groups** - User identity and group management
- **Custom Scripts** - Extend functionality with custom authentication and authorization scripts
- **Sessions** - Session management and policies
- **Casa** - Self-service credential management portal

Use the navigation menu to explore specific configuration topics.
