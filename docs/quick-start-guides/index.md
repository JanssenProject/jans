# Quick Start Guides

Get up and running quickly with the Janssen Project. Choose a quick start
guide below based on what you want to try:

## Janssen server Quick Start

Deploy Janssen Server using [Docker containers](../janssen-server/install/docker-install/quick-start.md) or [Helm based local Kubernetes environment](../janssen-server/install/helm-install/prerequisites/local.md). This is the fastest way to
get a working Janssen Server instance for testing and development.

## [Casa Quick Start](../casa/administration/quick-start.md)

Set up Jans Casa, a self-service web portal for end-users to manage
authentication and authorization preferences.

## [Cedarling Quick Start](../cedarling/quick-start/cedarling-quick-start.md)

Test authorization using the Cedarling, including role-based access control
with Cedar policies.
