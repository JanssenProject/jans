Jans Federation Vibe

OpenID Federation 1.0 implementation for Janssen

Available Endpoints

GET
/.well-known/openid-federation
Get entity configuration for a given issuer
GET
/federation/metadata
Get federation metadata
POST
/federation/validate-trust-chain
Validate trust chain for an entity
GET
/federation/trust-marks
Get trust marks for an entity
POST
/federation/issue-trust-mark
Issue a trust mark for an entity
GET
/federation/trust-mark-issuers
Get trust mark issuers

OpenID Federation 1.0 Specification

This implementation follows the OpenID Federation 1.0 specification.

Features

  • Entity Configuration management
  • Trust Mark issuance and validation
  • Trust Chain validation
  • Federation Metadata
  • JWT signing and verification using Nimbus JOSE JWT
